# VSCode 스니펫 설정하기

<Callout type="warn">
console.log를 매일 쓰고 있는데 이걸 계속 치다간 손목이 곧 망가질 것 같다라는 생각이 들었다.

cl만 검색해서 console.log를 완성하고 싶다..!

</Callout>

이전에 sass에 스니펫을 설정했었던 것 같은데 기억을 되돌려보자🤔

---

## 스니펫 설정 방법

<Steps>

<Step>
### Configure User Snippets 열기

`Cmd + SHIFT + P` (Windows/Linux: `Ctrl + SHIFT + P`) → **Configure User Snippets** 클릭

자신이 사용하는 언어를 클릭하자. 자바스크립트를 사용하니까 **javascript.json** 클릭!

</Step>

<Step>
### 스니펫 파일 확인

우리의 VSCode는 생각보다 친절하다

예시를 보고 만들고 싶은 스니펫을 만들어보자.

```jsx title="javascript.json (예시)"
// Place your snippets for javascript here. Each snippet is defined under a snippet name and has a prefix, body and
// description. The prefix is what is used to trigger the snippet and the body will be expanded and inserted. Possible variables are:
// $1, $2 for tab stops, $0 for the final cursor position, and ${1:label}, ${2:another} for placeholders. Placeholders with the
// same ids are connected.
// Example:
// "Print to console": {
// 	"prefix": "log",
// 	"body": [
// 		"console.log('$1');",
// 		"$2"
// 	],
// 	"description": "Log output to console"
// }
```

</Step>

<Step>
### 스니펫 속성 이해하기

<Callout type="info">
**스니펫 구조 설명**

- `prefix`는 호출할 **예약어**이고
- `body`에 설정하는 내용은 예약어 호출 시 **출력할 코드**
- 출력 코드 내부의 `$1`, `$2`로 설정된 부분은 포커스가 이동될 순서
- `$1`로 설정된 부분이 예약어 호출 시 처음으로 포커싱 될 위치이고 **tab** 키를 누르게 되면 `$2`로 포커싱이 이동한다.

</Callout>
</Step>

<Step>
### 스니펫 작성 및 저장

```json title="javascript.json"
"console.log print": {
    "prefix": "cl",
    "body": ["console.log($1) "],
    "description": "Log output to console"
  }
```

해당 내용을 추가하고 저장하면 끝!

</Step>

</Steps>

## 더 나아가기

### 자주 사용하는 스니펫 예시

```json title="javascript.json"
{
  "console.log print": {
    "prefix": "cl",
    "body": ["console.log($1)"],
    "description": "Log output to console"
  },
  "arrow function": {
    "prefix": "af",
    "body": ["const $1 = ($2) => {", "  $3", "}"],
    "description": "Arrow function"
  },
  "import React": {
    "prefix": "imr",
    "body": ["import React from 'react'"],
    "description": "Import React"
  },
  "useState Hook": {
    "prefix": "us",
    "body": ["const [$1, set${1/(.*)/${1:/capitalize}/}] = useState($2)"],
    "description": "React useState Hook"
  }
}
```

<Callout type="info">
**팁**: 스니펫에서 `${1/(.*)/${1:/capitalize}/}` 같은 문법을 사용하면 첫 번째 플레이스홀더의 값을 자동으로 변환할 수 있습니다!
</Callout>

### Windows/Linux 단축키

- **Windows/Linux**: `Ctrl + SHIFT + P`
- **Mac**: `Cmd + SHIFT + P`

이제 반복적인 코드 작성으로부터 자유로워지고 생산성을 높여보세요! 🚀
