## 문제

여러 개의 쇠막대기를 레이저로 절단하려고 한다. 효율적인 작업을 위해서 쇠막대기를 아래에서 위로 겹쳐 놓고, 레이저를 위에서 수직으로 발사하여 쇠막대기들을 자른다. 쇠막대기와 레이저의 배치는 다음 조건을 만족한다.

- 쇠막대기는 자신보다 긴 쇠막대기 위에만 놓일 수 있다.
- 쇠막대기를 다른 쇠막대기 위에 놓는 경우 완전히 포함되도록 놓되, 끝점은 겹치지 않도록 놓는다.
- 각 쇠막대기를 자르는 레이저는 적어도 하나 존재한다.
- 레이저는 어떤 쇠막대기의 양 끝점과도 겹치지 않는다.

아래 그림은 위 조건을 만족하는 예를 보여준다. 수평으로 그려진 굵은 실선은 쇠막대기이고, 점은 레이저의 위치, 수직으로 그려진 점선 화살표는 레이저의 발사 방향이다.

![쇠막대기와 레이저 배치](/images/content/algorithm-iron-rod-cutting.png)

이러한 레이저와 쇠막대기의 배치는 다음과 같이 괄호를 이용하여 왼쪽부터 순서대로 표현할 수 있다.

<Steps>

<Step>
### 레이저 표현

레이저는 여는 괄호와 닫는 괄호의 인접한 쌍 `()` 으로 표현된다. 또한, 모든 `()`는 반드시 레이저를 표현한다.

</Step>

<Step>
### 쇠막대기 표현

쇠막대기의 왼쪽 끝은 여는 괄호 `(` 로, 오른쪽 끝은 닫힌 괄호 `)` 로 표현된다.

</Step>

</Steps>

위 예의 괄호 표현은 그림 위에 주어져 있다.

쇠막대기는 레이저에 의해 몇 개의 조각으로 잘려지는데, 위 예에서 가장 위에 있는 두 개의 쇠막대기는 각각 3개와 2개의 조각으로 잘려지고, 이와 같은 방식으로 주어진 쇠막대기들은 총 17개의 조각으로 잘려진다.

쇠막대기와 레이저의 배치를 나타내는 괄호 표현이 주어졌을 때, 잘려진 쇠막대기 조각의 총 개수를 구하는 프로그램을 작성하시오.

### 입력

한 줄에 쇠막대기와 레이저의 배치를 나타내는 괄호 표현이 공백없이 주어진다. 괄호 문자의 개수는 최대 100,000이다.

### 출력

잘려진 조각의 총 개수를 나타내는 정수를 한 줄에 출력한다.

### 예제

## 문제 이해하기

<Callout type="info">
**핵심 포인트**

괄호 문제가 나오면 **스택(Stack)** 자료구조로 접근하기!

</Callout>

### 문제 분석

쇠막대기와 레이저의 배치를 나타내는 괄호가 주어졌을 때, 레이저로 잘린 쇠막대기 조각의 총 개수를 출력해야 합니다.

### 해결 전략

<Steps>

<Step>
### 문자열 순회

주어진 문자열을 하나씩 순회합니다.

</Step>

<Step>
### 여는 괄호 처리

여는 괄호 `(`를 만날 경우 → 모두 스택에 push합니다.

</Step>

<Step>
### 닫는 괄호 처리

닫는 괄호 `)`를 만날 경우:

- **레이저인가?** → 바로 전 인덱스 요소가 여는 괄호 `(`였다면 레이저
- **막대기의 끝인가?** → 바로 전 인덱스 요소가 여는 괄호가 아니라면 막대기의 끝

</Step>

<Step>
### 조각 개수 계산

- **레이저라면** → `stack.pop()` 후 `stack.length`를 더해주기
  - `stack.length`는 현재 막대기의 잘려진 개수
- **막대기의 끝이면** → `answer += 1`
  - 쇠막대기 한 개가 완성되면, 레이저로 잘리고 남은 쇠조각 1개만 남기 때문에 +1

</Step>

<Step>
### 결과 반환

for문이 종료되면 answer를 반환합니다.

</Step>

</Steps>

## 풀이 코드

```jsx title="solution.js"
function solution(s) {
  let answer = 0;
  let stack = [];

  for (let i = 0; i < s.length; i++) {
    if (s[i] === '(') stack.push(s[i]);
    else {
      stack.pop();
      if (s[i - 1] === '(') answer += stack.length;
      else answer += 1;
    }
  }

  return answer;
}

console.log(solution('()(((()())(())()))(())')); // 17
```

<Callout type="success">
**시간 복잡도**: O(n) - 문자열을 한 번만 순회하므로 효율적입니다.

**공간 복잡도**: O(n) - 최악의 경우 모든 괄호가 여는 괄호일 때 스택의 크기가 n이 됩니다.

</Callout>

### 코드 설명

1. **스택 초기화**: 쇠막대기를 추적하기 위한 빈 스택을 생성합니다.
2. **여는 괄호**: 새로운 쇠막대기의 시작이므로 스택에 추가합니다.
3. **닫는 괄호 + 직전이 여는 괄호**: 레이저이므로 현재 스택에 있는 모든 막대기가 잘립니다.
4. **닫는 괄호 + 직전이 닫는 괄호**: 막대기의 끝이므로 마지막 조각 1개를 추가합니다.

<Callout type="warn">
**주의사항**

이 문제는 `()`가 항상 레이저를 의미한다는 조건이 있어 간단하게 풀 수 있습니다. 조건을 정확히 파악하는 것이 중요합니다!

</Callout>
