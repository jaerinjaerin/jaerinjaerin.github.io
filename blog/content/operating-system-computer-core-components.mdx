중앙처리장치(CPU), 주기억장치(메모리), 보조기억장치, 입출력장치

## 메모리 (주기억장치)

![메모리 구조](/images/content/operating-system-computer-core-components/1.png)

메모리는 컴퓨터의 **작업 공간**입니다.

<Callout type="warn">
**메모리의 특징**

- **현재 실행**되고 있는 프로그램(프로세스)의 명령어와 데이터를 저장하는 공간
- 주소의 개념을 사용한다
- 전원이 꺼지면 저장된 내용을 잃는다 (휘발성)

</Callout>

### 주소 체계

메모리는 수많은 저장 공간으로 나뉘어져 있으며, 각 공간은 고유한 **주소(address)** 를 가집니다. 이를 통해 CPU는 필요한 데이터에 빠르게 접근할 수 있습니다.

---

## CPU (중앙처리장치)

![CPU 구조](/images/content/operating-system-computer-core-components/2.png)

CPU는 컴퓨터의 **두뇌**로, 프로그램의 명령어를 해석하고 실행합니다.

### CPU의 구성요소

<Callout>
**CPU 3대 구성요소**

1. **ALU (산술논리연산장치)**: 계산을 담당하는 계산기
2. **제어장치**: 제어신호를 보내고, 데이터를 읽고 쓰는 장치
3. **레지스터**: CPU 내부의 임시 저장 장치 (가장 빠른 메모리)

</Callout>

#### ALU (Arithmetic Logic Unit)

산술 연산(덧셈, 뺄셈 등)과 논리 연산(AND, OR, NOT 등)을 수행합니다.

#### 제어장치 (Control Unit)

- 명령어를 해석하고 실행 순서를 제어
- 메모리, 레지스터, ALU 등에 제어 신호를 보냄

#### 레지스터

- CPU 내부의 초고속 임시 저장 공간
- 메모리보다 훨씬 빠른 접근 속도

## 보조기억장치

![보조기억장치](/images/content/operating-system-computer-core-components/3.png)

보조기억장치는 데이터를 **영구적으로 보관**하는 저장소입니다.

<Callout type="info">
**보조기억장치의 특징**

- 용량이 크다 (메모리보다 훨씬 큼)
- 전원이 꺼져도 저장된 내용을 잃지 않는다 (비휘발성)
- 메모리보다 속도가 느리다

</Callout>

### 주요 보조기억장치

- **HDD (하드디스크 드라이브)**: 자기 디스크 기반
- **SSD (솔리드 스테이트 드라이브)**: 반도체 기반, HDD보다 빠름
- **USB 메모리, SD 카드** 등

---

## 입출력장치

![입출력장치](/images/content/operating-system-computer-core-components/4.png)

입출력장치는 컴퓨터와 외부의 **인터페이스** 역할을 합니다.

### 입력장치

사용자가 컴퓨터에 정보를 전달하는 장치

- 키보드, 마우스, 마이크, 웹캠 등

### 출력장치

컴퓨터가 처리한 결과를 사용자에게 전달하는 장치

- 모니터, 스피커, 프린터 등

## 메인보드와 시스템 버스

![메인보드](/images/content/operating-system-computer-core-components/5.png)

<Callout type='info'>
  핵심부품들은 모두 메인보드(마더보드)라는 판에 연결됩니다.
</Callout>

### 시스템 버스

메인보드에 연결된 부품들은 **버스(Bus)** 라는 통로로 서로 정보를 주고받을 수 있습니다.

<Callout>
**시스템 버스의 종류**

- **주소 버스**: 메모리 주소를 전달
- **데이터 버스**: 실제 데이터를 전달
- **제어 버스**: 제어 신호를 전달

</Callout>

여러 버스 중 네 가지 핵심 부품을 연결하는 버스가 **시스템 버스** 입니다.

---

## 정리

컴퓨터의 4가지 핵심 부품은 다음과 같은 역할을 합니다:

| 부품             | 역할                    | 특징                 |
| ---------------- | ----------------------- | -------------------- |
| **CPU**          | 명령어 해석 및 실행     | 컴퓨터의 두뇌        |
| **메모리**       | 실행 중인 프로그램 저장 | 빠르지만 휘발성      |
| **보조기억장치** | 데이터 영구 보관        | 용량이 크고 비휘발성 |
| **입출력장치**   | 사용자와의 인터페이스   | 데이터 입력/출력     |

이 4가지 부품이 **메인보드**의 **시스템 버스**를 통해 연결되어 하나의 컴퓨터 시스템을 구성합니다.
