코드스테이츠 프론트엔드 9주차 주간회고

한 주동안 무엇을 공부했나요?

이번 주에는 재귀에 대해서 열심히 공부했습니다. 매번 어렵다고 느꼈던 재귀인데 문제를 풀면서 재귀를 이런식으로 사용하는구나!라는 느낌을 받았습니다. 하지만 아직 언제 재귀를 사용해야할지에 대해선 감을 잡지 못한 것 같습니다.

UI/UX에 대해서도 공부했습니다. 이번 주엔 Figma를 사용하는 방법에 대해 배워보는 시간을 가졌습니다. 솔직히 말하면 Figma 클론하는 과제를 받았을 땐, 내가 피그마로 컴포넌트 만드는 것까지 배워야하나..?라는 생각이 들기도 했던 것 같습니다. 하지만 프론트엔드 개발자라면 디자이너와의 커뮤니케이션도 중요하기 때문에 배워놓자라는 마음을 먹었습니다. 공부를 하고나니 피그마로 컴포넌트를 제공하면서 리액트 컴포넌트 구조를 어떻게 만들지도 생각할 수 있었던 것 같습니다.

스터디에서는 리액트의 라이프사이클에 대해서 찾아보고 발표를 했습니다. 클래스형 컴포넌트를 배워본 적이 없어서 클래스형 라이프사이클 메서드에 대해서는 찾아볼 기회가 없었는데 스터디 덕분에 찾아보고 공부할 수 있는 기회가 된 것 같습니다.
