리액트에는 state라는 개념이 있고 state가 바뀌면 해당 컴포넌트와 하위 컴포넌트가 다시 렌더링하게 된다.

React State의 조건은?

  • 변경되지 않는건 state로 다룰 가치가 없다.

  • props로 전달받았다면 state가 아니다.

  • 계산 가능하면 state가 아니다.
